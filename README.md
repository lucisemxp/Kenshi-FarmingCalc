# Kenshi Farm Calculator

## Português

### Sobre o Programa
Calculadora de fazendas para o jogo Kenshi. O programa calcula a quantidade necessária de cada tipo de comida com base na população da sua base e, em seguida, calcula quantas plantações de cada tamanho são necessárias para produzir essa comida, considerando a produtividade do solo.

Atualmente o programa está disponível apenas em português, mas contribuições para tradução são bem-vindas.

### Como Usar
1. Preencha o número de cada raça/tipo de personagem na sua base
2. Preencha a produtividade do solo para cada tipo de planta (0-100%)
3. Clique em "Calcular"
4. Os resultados mostrarão:
   - Quantidade necessária de cada tipo de comida
   - Quantas plantações de cada tamanho são necessárias

### Arquivos do Projeto
- **backend.py**: Contém toda a lógica de cálculo, incluindo consumo por raça, valor nutricional dos alimentos, produtividade das plantações e funções de cálculo
- **main.py**: Interface gráfica construída com CustomTkinter, gerencia entrada de dados, exibição de resultados e interação com o usuário
- **requirements.txt**: Dependências necessárias para executar o programa
- **requirements-dev.txt**: Dependências adicionais para desenvolvimento (incluindo PyInstaller para criar executáveis)

## English

### About the Program
Farm calculator for the game Kenshi. The program calculates the required amount of each food type based on your base's population and then calculates how many farms of each size are needed to produce that food, considering soil productivity.

Currently the program is only available in Portuguese, but translation contributions are welcome.

### How to Use
1. Fill in the number of each race/type of character in your base
2. Fill in the soil productivity for each plant type (0-100%)
3. Click "Calcular" (Calculate)
4. Results will show:
   - Required quantity of each food type
   - How many farms of each size are needed

### Project Files
- **backend.py**: Contains all calculation logic, including consumption by race, food nutritional values, farm productivity, and calculation functions
- **main.py**: Graphical interface built with CustomTkinter, manages data input, result display, and user interaction
- **requirements.txt**: Dependencies needed to run the program
- **requirements-dev.txt**: Additional dependencies for development (including PyInstaller for creating executables)
